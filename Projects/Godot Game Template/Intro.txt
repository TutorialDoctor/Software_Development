# Godot Intro

## Initial steps
1. Create a project folder and asset folders
	- 2D assets:
		- levels
		- scripts
		- sounds
		- sprites
		- fonts
		- shaders
		- video
	
	- 3D assets:
		- levels
		- scripts
		- sounds
		- meshes
		- maps(textures)
		- fonts
		- shaders
		- video
		
2. Create a new project in the folder and open it
3. Create a root node (needed)
	- Node2D (2D levels)
	- Spatial (3D levels)
	- Control (GUI)
4. Save the scene (XML)
5. Set the main scene in project settings
